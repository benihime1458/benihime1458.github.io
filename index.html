brgIO

Full stack bouldering tracker at my home climbing gym. Boulderers at Bridges Rock Gym can sign up with their email (managed by Firebase Authentication) and track their progress on indoor bouldering problems. Frontend is a single page application built with React with Material-UI for styling. Backend is served using Express and connected to MongoDB Atlas. Currently data for bouldering problems are pulled from Google Sheets/CSV files but a user interface for manual updating and auto-updating the gym's problems is in progress. Currently deployed as a Docker container hosted on AWS EC2, but will be switching to Serverless architecture using Netlify-Lambda Functions to save on hosting costs.

Yelp Demo

RoR app featuring listings of restaurants that is entered by users managed with Devise gem. A location can be added and mapped on the Google Maps API. Images are uploaded using the CarrierWave gem and Amazon S3. ElasticSearch is utilized with Searchkick gem to add a layer convenience when searching for restaurants.
